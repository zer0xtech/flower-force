# Flower Force

A very simple project modelized and "coded" made on Delightex using their game engine and their language "CoBlocs".

It uses functions, manipulate the properties of the different objects in the map, change the animations of the characters, etc...

## Use

Go on https://www.delightex.com/ and click the execute button.
